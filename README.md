# Vityarthi_project-25bai10120

Project title- Digit to number printer
Overview of the project-The program first separates the numbers and forms a list to give out the number with the asked nth(input) digit in it. 
Features-The program is designed to perform conversions from numbers to number separated list and then back to the number, till 4 digit numbers (9999).
Technologies/tools used- if/elif statement, int to str, str to int, arithmetic operations(/,%,*,**,+), indexing, lists,variables
Instructions for testing- Beware of wrong indentation. And variables in vs code or system python software. 
The input for the program must be in the designated range only (1,3889).
Example:
n1(input)=100
the 100th digit as per separated list is tens place 5 in 55. 
output=
5
55
#this program works on the concept of lists and indexing, with alongside use of if and elif statements.
